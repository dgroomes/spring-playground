Showcasing various configuration features of Spring Boot and the Spring Framework.

## Instructions

* Use Java 11
* Run the program:
  * `./gradlew run`
* Alternatively, enable a Spring Boot profile named "dev" and run the program:
  * `SPRING_PROFILES_ACTIVE=dev ./gradlew run`
