# scheduling

A basic Spring Boot application that showcases scheduling using
the [`@Scheduled`](https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/scheduling/annotation/Scheduled.html)
annotation.

---

## Instructions

* Use Java 17
* Run the app:
    * `./gradlew run`
